Site Plan


Audience

The audience for this site will be children and adults who are looking for a fun and simple game to play. The game will be easy to understand and will be a fun way to pass the time.


Main Page

The Main page will hold the explanations for how to play the game and how to navigate the site


Game Page

The Game page will hold the game board and the game itself. It will have a refresh button at the bottom so users can restart when they want. Using javascript clicked elements will disappear and be replaced. If the user clicks on a tile with Pete then a game over pop up will show and propmt the user to refresh the page. Each time the page is created Pete will be placed randomly inside a tile.
